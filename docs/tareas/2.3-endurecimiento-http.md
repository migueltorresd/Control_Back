# Tarea 2.3 — Endurecimiento HTTP + versionado /api/v1

> Fase 2 — Seguridad perimetral · 🟠 Ítem 7 del plan · OWASP A01/A05/A07 · Depende de: 2.2 · Reglas: [README.md](README.md)

## Contexto / problema

- `app.enableCors()` sin opciones (`src/main.ts:8`) permite **cualquier origen**.
- Sin `helmet` (cabeceras de seguridad), sin rate limiting, sin límite de tamaño de body.
- La API no está versionada; el prefijo `/api/v1` se introduce ahora porque la visión arquitectónica prevé una API pública futura (landing/storefront) y versionar después de entregar obligaría a migrar al cliente.

## Archivos

- `src/main.ts`, `src/app.module.ts`
- `src/modules/auth/auth.controller.ts` (throttle del login)
- `src/config/env.validation.ts`, `.env.example`
- **Frontend**: `e:\proyectos\control-produccion\src\services\api.js` + nuevo `.env.example` del frontend

## Pasos

1. `pnpm add helmet @nestjs/throttler`.
2. En `main.ts`:
   - `app.setGlobalPrefix('api/v1')`
   - `app.use(helmet())`
   - `app.use(json({ limit: '1mb' }))` (de `express`)
   - CORS: `app.enableCors({ origin: configService.get('CORS_ORIGIN').split(',') })`.
3. Variable `CORS_ORIGIN` requerida en Joi y en `.env.example` (dev: `http://localhost:3000`; admite lista separada por comas).
4. `ThrottlerModule.forRoot` global: 100 requests/minuto por IP. Registrar `ThrottlerGuard` como `APP_GUARD`.
5. Login estricto: `@Throttle({ default: { limit: 5, ttl: 60000 } })` en `POST /auth/login`.
6. **Frontend (misma tarea — no romper el contrato)**: en `api.js`, `const API_URL = import.meta.env.VITE_API_URL ?? 'http://localhost:3001/api/v1'`. Crear `.env.example` del frontend con `VITE_API_URL=http://localhost:3001/api/v1`.

## Criterio de aceptación

- [ ] Rutas sin prefijo (`GET /vales`) → 404; con prefijo (`GET /api/v1/vales`) → funcionan.
- [ ] Desde un origen no listado, el navegador bloquea por CORS.
- [ ] 6º intento de login dentro de un minuto → 429.
- [ ] Respuestas incluyen cabeceras de helmet (p. ej. desaparece `X-Powered-By`).
- [ ] El frontend completo sigue funcionando contra `/api/v1`.
- [ ] Commit: `feat(http): api versioning, strict cors, helmet, rate limiting`.
